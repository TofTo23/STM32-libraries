# STM32 HAL libraries/programs (Nucleo board STM32-l476rg)
## choose a branch to see more information (description and .gif file)
1) step_motor - library to control step motor 28BYJ-48
2) chronometer - measurement of time (interrupt-driven)
3) ledIntensity - changing led intensity by pushing button (interrupt-driven)
4) rtos_v1 - three "parallel" tasks using freeRTOS
